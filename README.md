# Ejercitando Patrones de Diseño
**Practicando lógica de implementación de patrones de diseño habituales en la POO con PHP.**

### 1. ☠️ versus 🧟‍♂️ Crea un mini juego aplicando el patrón Factory en la creación de personajes.
Habrán dos personajes: *"Esqueleto"* y *"Zombi"* y cada personaje tendrá una lógica diferente en sus ataques y velocidad. La creación de los personajes dependerá del nivel de juego.
- En el nivel fácil se creará un personaje *"Esqueleto"*.
- En el nivel difícil se creará un personaje *"Zombi"*.

### 2. 🪟 ⇆ 📁 Crea un programa para compatibilizar archivos entre S.O. Windows aplicando el patrón Adapter.
Al compartir archivos de *Word*, *Excel* y *Power Point* de *Windows* 10 y querer abrirlos en *Windows* 7 surgen problemas de compatibilidad. Tenemos que trabajar con ambos sistemas, así que debes crear un programa donde Windows 10 acepte archivos independientemente que sean versiones anteriores.

### 3. ⚔️ + 🏹 Crea un mini juego con personajes usando armamento aplicando el patrón Decorator.
Con al menos dos personajes, al menos cierto(s) personaje(s) debe(n) poder elegir agregar armas a su arsenal.

### 4. 📨 －O－ 🤳 Crea un sistema de mensajería multiformato aplicando el patrón Strategy.
El sistema debe mostrar los mensajes en distintos tipos de salidas como por *Consola*, en formato *JSON* y archivo *TXT*.
